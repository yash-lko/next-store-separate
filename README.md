# Next Store Separate

A modern, high-performance e-commerce platform built with **Next.js 16**, **React 19**, and **TypeScript**. This project demonstrates best practices in full-stack web development with a focus on separation of concerns, scalability, and user experience.

## ✨ Features

- 🚀 **Next.js 16** - Latest React framework with App Router
- ⚛️ **React 19** - Modern component architecture
- 🎨 **Tailwind CSS** - Utility-first styling framework
- 🛒 **Zustand** - Lightweight state management
- 🎠 **React Slick** - Carousel component for product showcases
- 🔔 **React Hot Toast** - Toast notifications
- 🎵 **Sonner** - Alternative notification system
- 📦 **TypeScript** - Type-safe development
- 🧹 **ESLint** - Code quality and consistency
- 💨 **Lucide Icons** - Modern icon library

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yash-lko/next-store-separate.git
   cd next-store-separate
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## 🏗️ Architecture

This project follows a **component-based architecture** with:

- **Separation of Concerns** - Clear division between UI, logic, and state management
- **Type Safety** - Full TypeScript coverage for better developer experience
- **Scalability** - Modular structure for easy feature additions
- **Performance** - Optimized with Next.js best practices

## 🎨 Styling

The project uses **Tailwind CSS v4** for styling with PostCSS support, enabling:

- Utility-first CSS approach
- Responsive design
- Dark mode support (configurable)
- Custom theme customization

## 📦 State Management

**Zustand** is used for global state management, providing:

- Simple and intuitive API
- Lightweight bundle size
- TypeScript-first design
- Easy debugging
